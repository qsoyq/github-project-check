# 发布记录

本目录用于记录 `github-project-check` 规范文档和 Claude Code skills 的发布、分发和变更说明。

每次对外分发前建议确认：

- `README.md` 已更新。
- `.claude/skills/github-project-check/` 仍然自包含。
- `.claude/skills/github-checklist/` 仍然自包含。
- PR 模板、Issue 模板、CODEOWNERS 和 CI 检查通过。
- 变更风险和回滚方式已在 PR 中说明。

发布记录建议包含：

- 版本或日期。
- 变更摘要。
- 影响范围。
- 验证结果。
- 回滚方式。
